Known issues:
* You need to use VMware VIX 1.0.x (i.e. the version matched with your version of VMware Server 1.0.x).
  e.g. If you have VMware Server 1.0.4 you must use VIX 1.0.4
  On Windows, if you have VIX 1.1 set the library path to
    C:\Program Files\VMware\VMware VIX\server-1\32bit
  I have not tested on Linux
