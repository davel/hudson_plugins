Clausula "where" que sera utilizada pelo comando Report -raw -view TrackView.Utilizada para polling. Checa se houve mudanças na familia monitorada e caso positivo dispara a build.

Sao disponibilizadas 3 variaveis no contexto sao elas:

  • now
  • lastBuild
  • releases

E opcional. Se deixado em branco sera utilizda a clausula default: 

lastUpdate between $now and $lastBuild
and state = 'integrate' 
and releaseName in ($releases)
order by defectName
