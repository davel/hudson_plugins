Hudson can transform CppUnit test report XML format to JUnit XML format so it can be recorded by Hudson. When this option is configured, Hudson can provide useful information about test results, such as historical test result trend, web UI for viewing test reports, tracking failures, and so on.

To use this feature, first set up your build to run tests, then specify the path to CppUnit XML files in the Ant glob syntax, such as **/build/cppunit-reports/*.xml. Be sure not to include any non-report files into this pattern.
