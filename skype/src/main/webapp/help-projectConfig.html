this plugin is send message for Skype(http://www.skype.com/).
hudson -> hudson sype plugin -> skype4java(http://skype.sourceforge.jp/) -> skype.

Skype Build Status
send build status(change mood message)
Skype Build Error
send build error(match Error String) to Chat Name
Skype Test Error
send test error(match UnitTest. Now supported junit, nunit, flexunit, phpunit.) to Chat Name
