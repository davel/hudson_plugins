Run certain post-build actions once the join is achieved. The authors of the post-build actions did not have the join plugin in mind when writing their actions, so only a select few are shown here.

The join plugin can work with the paramaterized-trigger plugin, version 2.0 or later. Earlier versions will experience a NPE when started by the join plugin.

If there is a post-build action you would like to see supported, please file an issue for it. But do check the list of join plugin issues first.
