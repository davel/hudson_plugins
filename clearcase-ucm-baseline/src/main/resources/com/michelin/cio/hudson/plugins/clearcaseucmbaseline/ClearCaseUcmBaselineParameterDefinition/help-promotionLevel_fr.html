Optionnel — Niveau de promotion de la ligne de base ClearCase UCM qui sera utilisée par le projet lors d'un build.
Si aucun niveau de promotion n'est spécifié, alors, lors du démarrage d'un build, l'utilisateur pourra choisir parmi toutes les lignes de base ClearCase UCM du composant ClearCase UCM. Si un niveau de promotion est spécifié, alors l'utilisateur ne verra apparaître que les lignes de base ClearCase UCM correspondant à ce niveau.
