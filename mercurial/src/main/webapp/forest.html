When checked, the repository is considered the root of a forest. Make sure your Mercurial installation enables the Forest Extension. Known limitations in this release:
  • "Clean Build" will clean only the top-level repository.
  • Changelog entries will not note which repository a file is from, and hyperlinks to subrepositories will not work.
  • Pulling changes takes twice as long.
  • Snapshots are not yet supported; the remote repository must also have the extension enabled.
