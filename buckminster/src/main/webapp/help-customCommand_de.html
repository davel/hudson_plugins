Anstatt die Buckminster Kommandos direkt in der Job Konfiguration einzutragen können Sie auch eine Textdatei angeben die die Kommandos enthält.
Der Pfad der Textdatei ist relativ zum Arbeitsbereich des Build Jobs.

Falls Sie in die obere Text Box ebenfalls Kommandos eingetragen haben, so werden diese nur ausgefüllt wenn Sie dieses Feld leer lassen.
