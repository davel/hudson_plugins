Archiviert die gebaute Target Platform und veröffentlicht Sie für andere Projekte unter dem gegebenen Namen. Archivierte Dateien sind über die Hudson Seite abrufbar und Projekte die ein Buckminster Build Verfahren verwenden können aus einer Liste mit allen veröffentlichten Zielplattformen wählen. Normalerweise behält Hudson Artefakte eines Builds so lange wie den Build selsbt, aber wenn Sie nur eine Version benötigen und lieber Speicherplatz sparen, können Sie das tun.
