Von Zeit zu Zeit kann es vorkommen, dass Sie übergangsweise ein Projekt nicht automatisiert bauen lassen möchten. Beispiel: Sie stehen gerade Mitten in einer umfangreichen Migration und wissen, dass momentan neue Builds mit Sicherheit fehlschlagen werden. Oder: Das Projekt wird stündlich gebaut, aber Ihre CVS-Server wird die nächten 24 Stunden nicht verfügbar sein.

Wenn diese Option angewählt ist, werden keine neuen Builds dieses Projekts ausgeführt. Dies ermöglicht, neue Builds zu unterbinden, ohne die Abhängigkeiten zwischen Builds oder die Einstellungen für Benachrichtigungen ändern zu müssen.
