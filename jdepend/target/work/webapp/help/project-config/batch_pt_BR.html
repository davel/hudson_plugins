Executa um script do Windows para construir o projeto. O script será executado com o workspace como o diretório atual. O texto que você colocar na caixa de texto será executado como um arquivo de lote (.bat), e uma construção será considerada falha se ao final da execução %ERRORLEVEL% não for 0.

Se você já tem um arquivo de lote no SCM, você pode apenas digitar o caminho para o arquivo (novamente relativo ao diretório do workspace), e simplesmente executá-lo.
