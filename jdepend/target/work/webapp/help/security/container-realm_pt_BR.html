Usa o contêiner de servlet para autenticar os usuários, conforme definido pela especificação do servlet. Isto é históricamente o que o Hudson estava fazendo até a versão 1.163. Isto é útil principalmente para as seguintes situações:
  1. Você estava usando uma versão anterior a 1.164 e gostaria de manter seu comportamento.
  2. Você já configurou seu contêiner com domínio de segurança correto e prefere que o Hudson use-o. (algumas vezes o contêiner oferece uma documentação melhor ou implementações customizadas para conectar a domínio de usuários específico.)
