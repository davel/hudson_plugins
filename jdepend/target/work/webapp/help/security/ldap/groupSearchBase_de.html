Hudson ermittelt per LDAP die Liste der Gruppen, denen ein Benutzer angehört.

Dieses Feld enthält die Abfrage zur Identifikation der Organisationseinheit, welche Gruppen enthält. Die Abfrage ist fast immer "ou=groups", so dass Sie das Feld meistens leer lassen können und dieser Vorgabewert automatisch verwendet wird.
