Le plugin Piwik Analytics ajoute le code de tracking Piwik à toutes les pages de votre installation de Hudson
Piwik Analytics est un service de statistics gratuit sous licence GPL. Si vous n'avez pas d'instance de Piwik installée, vous pouvez récupérer les fichiers d'installation à l'adresse http://www.piwik.org Pour installer correctement le plugin Piwik Analytics vous aurez besoin d'un identifiant de site piwik, du nom de l'hôte et du repertoire d'installation de votre instance de Piwik.
