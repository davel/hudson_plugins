Piwik Analytics plugin that decorates all pages with Piwik tracking code.
Piwik Analytics is a statistics service provided free of charge under the GPL license. If you don't have a Piwik installed, you can get it at http://www.piwik.org
To be able to track your pages with Piwik, you will need a Site id, the Piwik server name and the Piwik install path.
