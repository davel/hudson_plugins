C'est votre identifiant de site piwik.
Dans l'interface d'administration de Piwik, quand vous ajoutez un nouveau site à monitorer, le code JavaScript à insérer contient cet identifiant. Exemple: dans ce code, l'identifiant de site Piwik est 5 ... var piwikTracker = Piwik.getTracker(pkBaseURL + "piwik.php", 5); ...
