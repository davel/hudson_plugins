Quand un build s'exécute sur un noeud esclave, copie un ensemble de fichiers depuis le noeud maître vers ce noeud esclave. Cela est utile quand, par exemple, certains fichiers nécessaires à la bonne exécution du build ne sont pas présents dans le SCM mais sont situés directement dans le workspace du projet.
Bien évidemment, aucune copie ne sera effectuée si le build s'effectue sur le noeud maître.
